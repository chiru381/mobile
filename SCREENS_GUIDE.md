# 📱 Screens & Navigation Reference\n\nComplete guide to all screens in the Memory App with accessibility paths.\n\n## 🔐 Authentication Screens\n\n### Splash Screen\n**File:** `app/splash.js`\n**Route:** `/splash`\n**Auto-Navigates:** Yes (after 3 seconds)\n**Navigation To:** `/login` or `/(auth)/login`\n\n**Features:**\n- Animated logo with scale and opacity\n- App name and tagline\n- Version display\n- Auto-redirect to login\n\n**Key Functions:**\n```javascript\n// Auto navigate after 3 seconds\nuseEffect(() => {\n  const timer = setTimeout(() => {\n    router.push('/(auth)/login')\n  }, 3000)\n}, [])\n```\n\n---\n\n### Login Screen\n**File:** `app/(auth)/login.js`\n**Route:** `/(auth)/login`\n**Accessible From:** Splash, Register (back button)\n\n**Features:**\n- 4 Login Methods:\n  1. MPIN (4-digit code)\n  2. Fingerprint\n  3. Face Recognition\n  4. Voice Recognition\n- Method switching\n- Remember me option\n- Forgot password link\n\n**Key Functions:**\n```javascript\nconst handleLogin = async () => {\n  // Validate input\n  // Call apiService.loginWithMpin()\n  // Navigate to home on success\n}\n\nconst handleBiometric = () => {\n  // Fingerprint/Face detection\n  // Navigate to home on success\n}\n```\n\n**Login Methods Flow:**\n```\nMPIN Login:\n  Input 4-digit code → Tap Login → Home\n\nFingerprint/Face:\n  Tap icon → System prompt → Home\n\nVoice:\n  Tap icon → Voice input → Home\n```\n\n---\n\n### Register Screen\n**File:** `app/(auth)/register.js`\n**Route:** `/(auth)/register`\n**Accessible From:** (Not in main flow yet, add link from login)\n\n**Features:**\n- Form fields:\n  - First Name\n  - Last Name\n  - Email\n  - Phone\n  - Password\n  - Confirm Password\n- Form validation\n- Error handling\n- Privacy terms checkbox\n\n**Key Functions:**\n```javascript\nconst handleRegister = async () => {\n  // Validate all fields\n  // Call apiService.register()\n  // Navigate to login on success\n}\n```\n\n**Form Validation:**\n- First Name: Required, min 2 chars\n- Email: Valid email format\n- Phone: Valid phone number\n- Password: Min 8 chars\n- Confirm Password: Must match\n\n---\n\n## 🏠 Main Tab Screens\n\n### 1.
