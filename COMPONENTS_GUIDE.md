# 🎨 Component Library Reference\n\nComplete guide to all reusable components in the Memory App.\n\n## AppButton\n\n**Location:** `components/AppButton.js`\n\nPrimary button component with multiple variants and sizes.\n\n### Props\n```javascript\n<AppButton\n  title=\"Click Me\"           // Button text (required)\n  onPress={() => {}}         // Callback function (required)\n  loading={false}            // Show loading spinner\n  disabled={false}           // Disable button\n  variant=\"primary\"          // 'primary' | 'secondary' | 'danger' | 'success'\n  size=\"md\"                  // 'sm' | 'md' | 'lg'\n  fullWidth={false}          // Stretch to full width\n  icon={<Icon />}            // Optional icon component\n/>\n```\n\n### Examples\n```javascript\n// Primary button\n<AppButton title=\"Login\" onPress={handleLogin} fullWidth />\n\n// Secondary variant\n<AppButton title=\"Cancel\" variant=\"secondary\" size=\"sm\" />\n\n// Danger with loading\n<AppButton \n  title=\"Delete\"\n  variant=\"danger\"\n  loading={isDeleting}\n  disabled={isDeleting}\n/>\n\n// With icon\n<AppButton \n  title=\"Download\"\n  icon={<Ionicons name=\"download\" size={18}
